Welcome to **Koota**, an ECS-based state management library optimized for real-time apps, games, and XR experiences. Use as much or as little as you need.

## Installation

```bash
npm i koota
```

If you prefer to jump into an example over reading docs, 
check out [Revade](https://github.com/pmndrs/koota/tree/main/examples/apps/revade) and our other [examples](https://github.com/pmndrs/koota/tree/main/examples).
